Steppy

Steppy is a tiny workflow engine for Java. It composes small, typed steps into flows that process data in sequence or in parallel.

The library focuses on a few core building blocks:

  • Step – a unit of work with typed input and output.
  • Flow – an executable pipeline of steps.
  • FlowBuilder – a fluent API for wiring steps together.
  • StepRepository – pluggable factory for obtaining step instances.

The rest of this guide walks through the major features with runnable snippets taken from the test suite.

  • Getting started – build your first flow.
  • Flows – execute steps sequentially or concurrently.
  • Branching & Nesting – route and compose subflows.
  • Streaming – process infinite sources.
  • State & Lifecycle – share data and hook into execution phases.
  • Configuration – access external configuration objects.
  • Dependency injection – integrate with Spring or CDI.
