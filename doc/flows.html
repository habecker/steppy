Flows

A flow is a chain of steps executed either sequentially or concurrently.

Sequential execution

Steps run one after another. Each step receives the previous result as input.

var flow = StaticFlowBuilderFactory
    .builder(String.class, String.class)
    .append(AppendAStep.class)
    .append(AppendBStep.class)
    .build();

Result<String> result = flow.invoke("" );
// result.getResult() == "AB"

Concurrent execution

Appending concurrent() makes the flow process a collection of inputs in parallel.

var flow = StaticFlowBuilderFactory
    .builder(String.class, String.class)
    .append(AppendAStep.class)
    .append(AppendBStep.class)
    .concurrent()
    .build();

var outputs = flow.invoke(List.of("", "C"));
// two Result<String> elements: "AB" and "CAB"

The returned collection mirrors the input order and captures success, failure or abort for each element.
