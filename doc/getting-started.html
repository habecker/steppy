Getting started

This quick example appends letters to a string to demonstrate how steps are defined, registered and combined into a flow.

1. Define the steps

public class AppendAStep implements Step<None, String, String> {
    @Override
    public String invoke(Context<None> ctx, String input) {
        return input + "A";
    }
}

public class AppendBStep implements Step<None, String, String> {
    @Override
    public String invoke(Context<None> ctx, String input) {
        return input + "B";
    }
}

2. Register steps and prepare the builder

StaticStepRepository.register(AppendAStep.class, AppendBStep.class);
StaticFlowBuilderFactory.initialize(Executors.newSingleThreadExecutor());

3. Build and run the flow

var flow = StaticFlowBuilderFactory
    .builder(String.class, String.class)
    .append(AppendAStep.class)
    .append(AppendBStep.class)
    .build();

Result<String> result = flow.invoke("");
System.out.println(result.getResult()); // prints "AB"

The same flow can process a collection of inputs or run concurrently by appending concurrent() before calling build(). Continue with the other chapters to explore advanced features.
