Dependency injection

Steps are created by a StepRepository. The provided StaticStepRepository registers classes or instances manually:

StaticStepRepository.register(AppendAStep.class, AppendBStep.class);
var flow = StaticFlowBuilderFactory
    .builder(String.class, String.class)
    .append(AppendAStep.class)
    .append(AppendBStep.class)
    .build();

For larger applications you can integrate with a DI container. Modules for CDI and Spring expose repositories that fetch steps from the application context, enabling constructor injection and other framework features.
